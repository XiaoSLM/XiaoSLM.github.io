Html学习笔记

1.Html简介

  • HTML 指的是超文本标记语言: HyperText Markup Language
  • HTML 不是一种编程语言，而是一种标记语言
  • 标记语言是一套标记标签 (markup tag)，学习Html就是学习标签
  • HTML 使用标记标签来描述网页

5.图像标签

图像标签：HTML 图像是通过标签 来定义的。默认包含两个属性：src和alt

其他属性：title 提示文本,鼠标放到图像上提示的文字，width 给图像设定宽度，height 给图像设定高度，宽高一般是通过CSS设置，border 给图像设定边框

插入图片的方式分为相对路径和绝对路径。一般使用相对路径，它包括同一级路径，下一级路径和上一级路径，也可以使用图片URL插入图片

6.视频和音频标签

Html是通过<video>标签来显示视频的

controls 播放控件 muted 静音
autoplay 自动播放 poster 预览图像
loop 循环播放
