1.Html简介

  • HTML 指的是超文本标记语言: HyperText Markup Language
  • HTML 不是一种编程语言，而是一种标记语言
  • 标记语言是一套标记标签 (markup tag)，学习Html就是学习标签
  • HTML 使用标记标签来描述网页

通过**!+回车**，编辑器可自动生成网页所需的必要代码，以后下文中出现的代码无特殊说明默认放在<body> </body>之间，即正文处。以下是详细解析：

3.标题与段落标签

标题标签：HTML 标题是通过<h1> - <h6> 标签来定义的，标题文字会加粗（默认）显示，且每行只会显示一个，<h1>一级标签</h1>最好每个页面只使用一次。

段落标签：HTML 段落是通过标签 <p> 来定义的

4.文本格式化标签及注释标签

显示结果：

我是加粗的文字 我是加粗的文字
我是倾斜的文字 我是倾斜的文字
我是删除线 我是删除线
我是下划线的文字 我是下划线的文字

注释标签：HTML 图像是通过标签 来定义的.

5.图像标签

图像标签：HTML 图像是通过标签<img> 来定义的。默认包含两个属性：src和alt

其他属性：title 提示文本,鼠标放到图像上提示的文字，width 给图像设定宽度，height 给图像设定高度，宽高一般是通过CSS设置，border 给图像设定边框

插入图片的方式分为相对路径和绝对路径。一般使用相对路径，它包括同一级路径，下一级路径和上一级路径，也可以使用图片URL插入图片

6.视频和音频标签

Html是通过<video>标签来显示视频的

主要属性 作用 主要属性 作用
src 视频地址 width/ height 宽度/高度
controls 播放控件 muted 静音
autoplay 自动播放 poster 预览图像
loop 循环播放

Html是通过<audio>标签来显示音频,属性与<video>基本相同

7.超链接标签

HTML使用标签<a> 来设置超文本链接,在标签 <a> 中使用了 href 属性来描述链接的地址

target：定义链接的打开方式

  • _blank: 在新窗口或新标签页中打开链接。

  • _self: 在当前窗口或标签页中打开链接（默认）。

  • _parent: 在父框架中打开链接。

  • _top: 在整个窗口中打开链接，取消任何框架。

id：用于链接锚点，通常在同一页面中跳转到某个特定位置

8.列表标签

无序列表使用 <ul> 标签

有序列表使用 <ol> 标签

自定义列表以 <dl> 标签开始，每个自定义列表项以 <dt> 开始。每个自定义列表项的定义以 <dd> 开始

9.表格标签

HTML 表格由 <table>标签来定义。每个表格均有若干行（由 `` 标签定义），每行被分割为若干单元格（由 标签定义），表格可以包含标题行（）用于定义列的标题。同时可以加 `来使表格结构更清晰

10. 表单

HTML 表单用于收集用户的输入信息。

主要属性 作用 主要属性 作用
action 定义了表单数据提交的目标 URL select/option 用于创建下拉列表 /用于定义下拉列表中的选项
label 用于为表单元素添加标签，提高可访问性，点击标签可聚焦输入框 method 定义了提交数据的 HTTP 方法
input 创建文本输入框、密码框、单选按钮、复选框 type 定义了输入框的类型
textarea 文本域标签 file 文件域标签

11.HTML 区块元素

HTML

元素是块级元素，它可用于组合其他 HTML 元素的容器。一行只有一个

HTML 元素是内联元素，可用作文本的容器,一行可以有多个
